D
disjoint anchor #intent

An IML anchor that encompasses multiple, separate stretches of content as a single composite unit.

For example, suppose an English teacher wants to illustrate the rhyme scheme in Shakespeare’s Sonnet 29. She might define the endings of rhyming lines to be part of a single disjoint anchor. When she later links to such an anchor, she is able to point to all items that share a common anchor ID as a unit. intent markup language #intent

The subset of intent language that specializes in rich exposition. IML is a lightweight markup language with a syntax that resembles markdown. It is also like markdown in that it is designed to be transformed into HTML for pretty rendering.

version 1, commit 09b4777, created 2021-11-06, contributors Daniel Hardman

intent programming language #intent

The subset of intent language that emphasizes formal structures and instructions, and that provides the primary input that the compiler transforms into machine code. IPL is a hybrid general-purpose programming language that incorporates the best of functional, declarative, and object-oriented paradigms. It shares some inspirations with natural-language programming and Structured English.

mark #intent

An annotation that adds or subtracts semantics from the construct it describes. The name "mark" comes from the concept of linguistic markedness.

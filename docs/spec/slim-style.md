# Slim Style

Slim style is a set of conventions used by writers and editors. It embodies choices about punctuation, capitalization, and citation, and in those areas it proposes to override recommendations from venerable sources such as <cite>APA</cite>, <cite>Chicago Manual of Style</cite> or <cite>MLA</cite>. However, the scope of Slim Style is smaller, so it is not a wholesale replacement.

## Why?

Existing style guides originated in an era of typewriters, physical media, and publication as a specialized industry. Today, reading and writing both differ in important ways. We use a bewildering array of tools, and their features are both powerful and inconsistent: word processors, email clients, spreadsheets, social media interfaces, forms on websites, SMS, voice-to-text... Copy-paste is ubiquitous, but so is lossy transfer from one context to another. We expect to change the window size or the visual theme of text on demand. Hyperlinks and hash tags and mentions abound. Crowdsourced wikis and content editing make attribution harder. Leet, globalization, and online translation services increase our exposure to different registers and a broader spectrum of language abilities. Formalized written formats such as markdown and source code matter to a growing proportion of the public.

Old style guides all show progress at reacting to these pressures, but they are changing too slowly. We need a quantum leap.

## Principles

Slim Style aims to focus humans on distilled meaning. This should be representable in any tool, regardless of its feature set. Make the meaning clear, even to dumb computers, and let computers make things as fancy as circumstances demand.

## Rules

### Punctuation
1. Nest accurately.
Use quotes to delimit exactly and only what was said elsewhere. Don't use someone else's sentence boundary as your own.
    * Alice said, "Are you going to the store?". -- The quote is a question, and we want to convey that, so the question mark goes inside the close quote. The period outside the quote shows that the writer is choosing to end the sentence after reporting Alice's utterance.
    * Alice said, "Are you going to the store?", but I ignored her. -- Same analysis of the quote, but now the writer wants to continue 
3. If it's a complete sentence, punctuate it as such. If it's not, don't.
4. Put a single space after end-of-sentence punctuation.
5. Represent an em dash as two hyphens, and let software make it look pretty if it wants.
